# موقع السيرة الذاتية الإلكترونية — مفيد الشميري

مشروع React + Vite بتصميم عربي RTL، جاهز للرفع إلى GitHub والنشر على Vercel.

## تشغيل المشروع على الكمبيوتر

```bash
npm install
npm run dev
```

ثم افتح الرابط الذي يظهر لك في المتصفح.

## بناء نسخة الإنتاج

```bash
npm run build
```

## رفع المشروع إلى GitHub

1. أنشئ Repository جديدًا على GitHub.
2. ارفع جميع ملفات المشروع إلى المستودع.
3. اربط المستودع مع Vercel.
4. اختر مشروع Vite وسيتم النشر تلقائيًا.

## تعديل بيانات السيرة

افتح الملف:

`src/main.jsx`

وعدّل بيانات `profile` والخبرات والدورات والمهارات.

## ملاحظة

زر "حفظ / طباعة السيرة" يستخدم نافذة الطباعة في المتصفح، ويمكن اختيار "Save as PDF" لحفظ السيرة كملف PDF.
